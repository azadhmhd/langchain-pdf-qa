This repo lets you use a local PDF/text file to ask questions and generate asnwers.

## How to use

1. Add a .env file:

```
OPENAI_API_KEY=
```

2. Run `npm i`
3. Run `nm run dev`

You will see a response in your console for the Microsoft 10-K report
